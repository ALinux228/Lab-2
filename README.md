# Задание 1

## Задача 1

### List.map (на основе данного списка получить новый список)

Получить список из последних цифр чисел, содержащихся в исходном списке

### Алгоритм решения

1. Для получения последней цифры используется функция abs x % 10, которая:

      - Берет модуль числа (abs) для корректной обработки отрицательных чисел

      - Находит остаток от деления на 10, который и является последней цифрой

2. Применить эту функцию к каждому элементу списка с помощью List.map, которая вернёт список целых чисел

# Задание 2

## Задача 1

### List.fold

Найти сумму тех элементов списка, которые оканчиваются на заданную цифру.

### Алгоритм

1. Запросить у пользователя цифру для поиска (от 0 до 9)

2. Реализовать проверку корректности ввода:

	- Проверка, что число находится в диапазоне от 0 до 9

	- При неверном вводе - повторный запрос

3. Для нахождения суммы используется List.fold

4. Для каждого элемента проверяется, оканчивается ли он на заданную цифру

	- Если да - элемент добавляется к аккумулятору

	- Если нет - аккумулятор остается без изменений
